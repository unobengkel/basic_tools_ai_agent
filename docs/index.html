Pengenalan

Program ini adalah contoh praktis penggunaan Function Calling (Tools) pada AI DeepSeek. Tool Use memungkinkan AI untuk berinteraksi dengan dunia nyata melalui fungsi-fungsi Python yang Anda definisikan.

Berbeda dengan chat biasa, AI bisa "memutuskan" untuk memanggil fungsi (seperti mengecek harga saham atau cuaca) ketika mendeteksi bahwa pertanyaan pengguna membutuhkan data eksternal.

Kenapa Ini Penting?

AI seringkali memiliki pengetahuan yang terbatas pada tanggal tertentu. Dengan Tools, Anda memberikan AI kemampuan untuk:

  • Mengambil data terbaru dari API.
  • Melakukan perhitungan kompleks.
  • Berinteraksi dengan database atau file sistem.

Setup & Instalasi

Langkah 1: Instalasi Library

DeepSeek menggunakan API yang kompatibel dengan OpenAI, jadi Anda cukup menginstal library resmi OpenAI:

pip install openai

Langkah 2: Konfigurasi API Key

Anda harus memiliki API Key dari DeepSeek Platform. Setel key tersebut sebagai environment variable untuk keamanan:

# Windows (PowerShell)
$env:DEEPSEEK_API_KEY = "sk-..."

# Linux / Mac
export DEEPSEEK_API_KEY="sk-..."

Cara Penggunaan

Ikuti langkah berikut untuk menjalankan skrip contoh:

  1. Buka terminal di folder project belajar_tools_ai.
  2. Jalankan skrip utama: 
    python deepseek_tools.py

Apa yang Terjadi di Balik Layar?

Saat Anda bertanya "Berapa harga saham Apple?":

  1. Model DeepSeek melihat daftar tools yang tersedia.
  2. AI menjawab dengan pesan khusus untuk memanggil fungsi get_stock_price.
  3. Skrip Python menerima permintaan itu, mengeksekusi fungsi lokal, dan mengirim hasilnya balik ke AI.
  4. AI memberikan jawaban akhir berdasarkan data dari fungsi tersebut.

Metode Belajar

Untuk menguasai penggunaan AI Tools, kami menyarankan urutan belajar berikut:

1. Perhatikan Schema JSON

Lihat variabel tools di dalam kode. Pahami bagaimana description sangat berperan penting agar AI tahu kapan harus menggunakan fungsi tersebut.

2. Eksperimen Parameter

Coba ganti input pengguna dengan simbol saham yang tidak ada dalam daftar (misal: "GOTO"). Perhatikan bagaimana logika else di Python menangani hal tersebut.

3. Tambahkan Tool Baru

Tantangan: Buat fungsi baru get_crypto_price dan tambahkan ke daftar tools. Ini adalah cara terbaik untuk memvalidasi pemahaman Anda.
